# 세입자 AI 토글 및 매물 문의 영역 정리 설계

## 목표

- 세입자 화면의 원형 AI 버튼 하나로 AI 패널을 열고 닫을 수 있게 한다.
- 매물 상세에서 전화 문의 진입점과 로그인 없는 문의 안내 문구를 제거한다.
- 기존 문자 문의와 3D 관련 기능은 유지한다.

## 세입자 AI 동작

- AI 패널이 닫혀 있으면 원형 AI 버튼은 패널을 연다.
- AI 패널이 열려 있어도 원형 AI 버튼을 숨기지 않으며, 다시 누르면 패널을 닫는다.
- 패널 헤더의 기존 닫기 버튼은 보조 닫기 수단으로 유지한다.
- 어느 닫기 수단을 사용하든 활성 음성 연결을 먼저 종료하고 패널 상태를 닫힘으로 바꾼다.
- 원형 버튼의 접근성 이름과 상태는 현재 패널 상태를 반영한다. 닫힘 상태는 `AI 생활 도우미 열기`, 열림 상태는 `AI 생활 도우미 닫기`로 노출하고 `aria-expanded`도 동기화한다.

## 매물 상세 문의 영역

- 데스크톱 우측 CTA에서 `로그인 없이 문의 가능 · 채팅으로 바로 연결` 문구를 제거한다.
- 데스크톱 우측 CTA에서 `전화` 버튼을 제거한다.
- 하단 고정 문의 바에서 같은 안내 문구와 전화 버튼을 제거한다.
- `문자로 문의하기`, 3D 도면/둘러보기, 1인칭 체험 동작은 그대로 유지한다.
- 전화 버튼 제거 후 남은 하단 버튼들이 사용 가능한 너비를 채우도록 기존 토큰 기반 CSS 그리드 열을 조정한다.
- 더 이상 쓰이지 않는 전화 아이콘 import와 전용 스타일은 제거한다.

## 테스트 및 검증

- 정적 회귀 테스트에 원형 AI 버튼이 열린 상태와 닫힌 상태를 모두 처리하고, 열린 상태에서도 숨겨지지 않는다는 조건을 추가한다.
- 매물 상세 소스에 제거 대상 안내 문구와 전화 문의 버튼이 없다는 조건을 추가한다.
- 하단 문의 바의 그리드가 전화 버튼 없는 구조와 일치하는지 검증한다.
- 대상 web 테스트를 먼저 실패시키고 구현 후 통과시킨다.
- 마지막에 전체 web 테스트와 `bash scripts/verify.sh`를 실행한다.

## 범위 밖

- 문자 문의의 로그인 또는 채팅 연결 정책은 변경하지 않는다.
- 세입자 AI 대화 내용과 세션 보존 정책은 변경하지 않는다.
- 3D 투어 및 1인칭 체험의 노출 조건은 변경하지 않는다.
