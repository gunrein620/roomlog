# 가구 카테고리 가로 스크롤 표시 설계

## 목표

가구 카탈로그의 카테고리 칩이 패널 너비를 넘을 때 사용자가 남은 카테고리가 있다는 사실과 좌우 이동 방법을 즉시 알 수 있게 한다.

## 동작

- 카테고리 칩은 기존처럼 한 줄로 유지한다.
- 카테고리 영역 바로 아래에 항상 보이는 얇은 가로 스크롤바를 표시한다.
- 마우스로 스크롤바를 끌거나, 터치 화면에서 좌우로 밀어 이동할 수 있다.
- 선택된 카테고리 스타일과 필터 동작은 변경하지 않는다.
- 세로 스크롤과 가구 목록 영역에는 영향을 주지 않는다.

## 구현 범위

- `listing-tour-furniture-category-tabs`의 기존 `overflow-x: auto`를 유지한다.
- Firefox와 Chromium 계열 브라우저에서 스크롤 트랙과 손잡이가 식별되도록 스타일을 추가한다.
- 손잡이는 현재 UI의 회색 계열을 사용하고, 트랙은 밝은 배경으로 구분한다.
- 좁은 화면에서도 칩이 줄바꿈되지 않도록 각 버튼의 `flex: 0 0 auto`를 유지한다.

## 검증

- 소스 계약 테스트에서 가로 오버플로와 브라우저별 스크롤바 스타일을 확인한다.
- 테스트가 기존 스타일에서 실패하는 것을 확인한 뒤 최소 CSS를 추가한다.
- 카테고리가 잘리는 폭에서 스크롤바가 보이고 마지막 카테고리까지 이동되는지 브라우저로 확인한다.
