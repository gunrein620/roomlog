# 관리인 메시징 연결 업무 안내 제거 설계

## 목표

관리인 메시징 상세 화면의 맥락 카드에서 다음 안내 문구를 제거한다.

`연결된 업무: <업무 ID> · 임차인에게도 같은 대화가 표시됩니다.`

## 변경 범위

- `ContextCard`의 배지 목록은 유지한다.
- `일반 문의` 등 `thread.contextLabel`로 표시되는 제목은 유지한다.
- 제목 아래에서 연결 업무와 임차인 노출 여부를 설명하는 안내 영역만 제거한다.
- 메시지 타임라인, 답장 입력, 추가 요청 및 AI 답장 영역의 동작은 변경하지 않는다.

## 구현 방법

`apps/web/src/app/manager/messaging/04/page.tsx`의 `ContextCard`에서 대상 안내 문구를 감싼 마지막 `<div>`만 삭제한다. CSS 숨김이나 조건부 렌더링은 사용하지 않는다.

## 검증

- 관리인 메시징 상세 페이지 소스에 `연결된 업무:` 문구가 없는지 확인한다.
- 같은 소스에 `임차인에게도 같은 대화가 표시됩니다.` 문구가 없는지 확인한다.
- `thread.contextLabel ?? "일반 문의"` 제목 렌더링이 유지되는지 확인한다.
- 관리인 메시징 관련 web 단위 테스트와 전체 web 테스트를 실행한다.
