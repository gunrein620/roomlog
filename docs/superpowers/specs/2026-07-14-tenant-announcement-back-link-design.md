# 세입자 공지 목록 뒤로가기 라우트 변경 설계

## 목표

`/tenant/messaging/02` 공지 목록 헤더의 뒤로가기 버튼을 누르면 `/living`으로 이동하게 한다.

## 변경 범위

- `AnnouncementListPage.tsx`의 뒤로가기 `Link`에서 `href`만 `/tenant/home/00`에서 `/living`으로 변경한다.
- 기존 아이콘, 스타일, `aria-label="세입자 홈으로 돌아가기"` 문구는 변경하지 않는다.
- 공지 상세 화면과 다른 세입자 화면의 뒤로가기 동선은 변경하지 않는다.

## 테스트와 검증

1. 공지 목록 계약 테스트에 뒤로가기 링크가 `/living`인지 검증하는 실패 테스트를 먼저 추가한다.
2. 링크의 `href`만 변경한 뒤 집중 계약 테스트와 web 검증을 실행한다.
3. Docker web 이미지를 재빌드하고 실제 버튼 클릭 후 주소가 `/living`인지 확인한다.

## 비범위

- 브라우저 방문 기록 기반 `router.back()` 동작
- 접근성 문구 변경
- 공용 라우트 상수 및 인프라 설정 변경
