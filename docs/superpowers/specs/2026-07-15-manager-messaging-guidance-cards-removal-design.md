# 관리인 메시징 안내 카드 제거 설계

## 목표

관리인 메시징 상세 화면의 우측 영역에서 추가 요청, 맥락 톤, 음성 답장 확인 안내 카드를 표시하지 않는다.

## 변경 범위

- `추가 요청` 안내 카드를 제거한다.
- 일반 문의에서 표시되는 `맥락 톤` 안내 카드를 제거한다.
- 결제 문의에서 같은 위치에 표시되는 `청구 맥락 톤 가드` 안내 카드도 제거한다.
- `음성 답장 확인 1스텝` 안내 카드를 제거한다.
- `사진 요청`, `설명 요청` 버튼은 유지한다.
- `음성 받아쓰기 → 텍스트 확인` 버튼은 유지한다.
- `AI 답장 초안` 카드와 `초안 적용` 버튼은 유지한다.
- 메시지 타임라인과 답장 입력 영역은 변경하지 않는다.

## 구현 방법

`apps/web/src/app/manager/messaging/04/page.tsx`에서 대상 `NoticeCard`와 맥락별 조건부 카드 블록만 삭제한다. CSS 숨김이나 새로운 조건부 렌더링은 추가하지 않는다. 맥락 톤 블록 제거로 사용되지 않게 되는 `isPayment` 지역 변수도 함께 삭제한다.

## 검증

- 세 안내 제목과 결제 대체 안내 제목이 상세 페이지 소스에 없는지 확인한다.
- 사진 요청, 설명 요청, 음성 받아쓰기 버튼이 유지되는지 확인한다.
- AI 답장 초안과 초안 적용 버튼이 유지되는지 확인한다.
- 관리인 메시징 상세 집중 테스트를 실행한다.
